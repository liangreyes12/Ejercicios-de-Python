# BLOQUE 1: Control de flujo y condiciones (1–10) 🐍

Este directorio contiene 10 ejercicios de programación en Python que cubren diversos temas de lógica de programación, validación de entrada, uso de módulos externos y manipulación de datos.

## Estructura de Archivos

### Archivos Principales (1-10)

| # | Archivo | Descripción |
|---|---------|-------------|
| 1 | `1PositivoNegativoCero.py` | Determina si un número es positivo, negativo o cero |
| 2 | `2parImpar.py` | Verifica si un número es par o impar |
| 3 | `3mayorMenor.py` | Encuentra el número mayor entre n números ingresados |
| 4 | `4añoBisiciesto.py` | Determina si un año es bisiesto |
| 5 | `5login.py` | Sistema de inicio de sesión con validación de usuarios |
| 6 | `6multiplos3y5.py` | Verifica si un número es múltiplo de 3 y 5 |
| 7 | `7producto.py` | Sistema de tienda con descuentos escalonados |
| 8 | `8notas.py` | Clasificación de notas académicas |
| 9 | `9triangulo.py` | Análisis y clasificación de triángulos |
| 10 | `10anagrama.py` | Detector de anagramas |

### Módulos Externos

Los ejercicios 7, 8 y 9 hacen uso de módulos externos ubicados en la carpeta `MODULOS/`:

- **`MODULOS/supermercado.py`**: Módulo para gestión de tienda
  
- **`MODULOS/notas.py`**: Módulo para gestión de notas académicas
  
- **`MODULOS/calculoTriangulo.py`**: Módulo para clasificación de triángulos

---

## Requisitos del Sistema

- Python 3.x
- No requiere librerías externas adicionales (usa bibliotecas estándar)

## Ejecución de Todos los Ejercicios

Para ejecutar todos los ejercicios, debes estar en la carpeta principal `EJERCICIOS\` y digitar el comando:

```bash
# Windows
python -m <NombreDeBloque>.<NombreDearchivo>.py

# Linux
python3 -m <NombreDeBloque>.<NombreDearchivo>.py
```

## Notas Adicionales

1. **Manejo de Errores:** Todos los ejercicios incluyen manejo de excepciones básico
   
2. **Validación de Entrada:** La mayoría valida que los datos ingresados sean del tipo correcto
   
3. **Interfaz:** Los programas son de consola (CLI)
   
4. **Los módulos externos** deben estar en la carpeta `MODULOS/` para que los ejercicios 7, 8 y 9 funcionen correctamente
