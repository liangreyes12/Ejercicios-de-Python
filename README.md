# EJERCICIOS DE PYTHON DE 0 HASTA EXPERTO 🐍👨‍💻

## 📜 Descripción

Este repositorio esta diseñado para mostrar el aprendizaje desde cero hasta un nivel de experto con el manejo del lenguaje de programación **Python**.

Esta dividido en una estructura de carpetas según el nivel y dificultad de los ejercicios ***(División por bloques)***:

- Cada bloque cuenta con ***(10)*** ejercicios

- Cada bloque cuenta con un archivo ***README.md*** con información sobre lo que abarca.
  
- Cada archivo ***(.py)*** dentro de los bloques tendrán una documentación breve sobre los ejercicios y su resolución

- Algunos ejercicios hacen uso de **módulos externos**,  creados para facilitar la codificación y buenas prácticas de desarrollo. Los módulos externos los podrás encontrar en la carpeta: ***"MODULOS"***

- Los ejercicios hacen uso de **los modulos externos**, para poder ejecutarse en la terminal debes estar en ***(la carpeta principal)*** `EJERCICIOS\`  y usar el **comando**:
  
```bash
# Windows
python -m <NombreDeBloque>.<NombreDearchivo>.py

# Linux
python3 -m <NombreDeBloque>.<NombreDearchivo>.py
````
